# NIA (Network Intelligence Analytics) 제품 문서

## 프로젝트 개요

**NIA(Network Intelligence Analytics)**는 네트워크 모니터링, 관리 및 분석을 위한 종합 플랫폼입니다. 마이크로서비스 아키텍처 기반으로 구축되어 대규모 네트워크 환경의 성능 최적화, 장애 진단 및 예측 유지보수를 지원합니다.

### 핵심 목표

- 네트워크 성능 실시간 모니터링 및 분석
- IP, 광학망, 무선망 등 다중 도메인 통합 관리
- AI/머신러닝 기반 근본 원인 분석(RCA) 및 예측 유지보수
- SDN/EMS 연동을 통한 자동화된 네트워크 제어
- 생산 환경의 안정성과 효율성 극대화

## 대상 사용자

- **네트워크 운영팀(NOC)**: 네트워크 실시간 모니터링, 알람 관리, 장애 대응
- **네트워크 엔지니어**: 트래픽 분석, 성능 튜닝, 용량 계획
- **인프라 담당자**: 시스템 성능 분석, 리소스 최적화
- **보안팀**: 이상 탐지, 트래픽 분석, 보안 이벤트 추적

## 핵심 기능

### 1. 네트워크 성능 모니터링

**IP 네트워크 모니터링**
- 라우터, 스위치, 호스트 등 네트워크 장비의 성능 실시간 모니터링
- 인터페이스 상태, 대역폭 사용률, 패킷 손실률 추적
- SNMP/syslog 기반 장비 데이터 수집 및 분석

**광학 네트워크 모니터링**
- 광학 네트워크 장비(ROADM, 광증폭기 등) 성능 모니터링
- 신호 품질, 광신호 레벨 등 광학 파라미터 추적
- TL1 프로토콜 기반 광학 장비 제어 및 관리

**무선 네트워크 모니터링**
- 무선 접속점(AP), 기지국 등의 무선 자산 모니터링
- 신호 강도, 접속 사용자, 데이터 처리량 추적

### 2. 트래픽 분석

**sFlow 기반 트래픽 분석**
- 네트워크상의 실제 트래픽 패턴을 샘플링하여 분석
- 상위 N개 대화자(Top-N Talkers), 프로토콜 분포 파악
- 이상 트래픽 탐지 및 보안 위협 식별

**트래픽 시각화**
- 계층별(L2-L7) 트래픽 분포도
- 시계열 트래픽 추이 그래프
- 네트워크 토폴로지상의 트래픽 흐름 시각화

### 3. 근본 원인 분석(RCA)

**지능형 RCA 엔진**
- 머신러닝 기반 이상 탐지 및 근본 원인 추적
- 다양한 네트워크 메트릭 간의 상관관계 분석
- 복합 장애 상황의 원인 파악 및 가설 생성

**AI 기반 분석**
- 시계열 데이터 분석을 통한 이상 패턴 식별
- 예측 모델을 통한 미래 문제 예측
- 운영 데이터의 의미 있는 통찰력 도출

### 4. 예측 유지보수

**광학 네트워크 예측 유지보수**
- 광학 품질 저하 추세 분석
- 장비 장애 예측 및 선제적 유지보수
- 신뢰성 및 가용성 향상

**성능 기반 유지보수**
- 성능 메트릭 기반 장비 건강도 평가
- 유지보수 필요 시점 사전 예측
- 비용 절감 및 다운타임 최소화

### 5. 알람 수명 관리

**지능형 알람 처리**
- 알람 발생, 확대, 수신, 처리, 종료 전 과정 관리
- 중복 알람 자동 필터링 및 통합
- 알람 심각도 자동 평가 및 우선순위 매김

**알람 연계**
- 알람 조건에 따른 자동 장애 격리
- 네트워크 구성 변경 자동 실행
- 외부 시스템(EMS, 티켓 시스템) 연동

### 6. EMS/SDN 외부 시스템 연동

**SDN 통합**
- OpenFlow, NETCONF 등 표준 프로토콜 기반 SDN 제어
- 네트워크 정책 자동 적용 및 수정
- 트래픽 엔지니어링 자동화

**EMS 연동**
- 기존 EMS 시스템과의 통합
- 양방향 데이터 동기화
- 통합 대시보드를 통한 통일된 관리

### 7. 클러스터 조율

**고가용성 클러스터**
- 다중 노드 클러스터 구성으로 높은 가용성 보장
- 자동 페일오버 및 로드 밸런싱
- 실시간 데이터 복제 및 동기화

**분산 처리**
- 마이크로서비스 기반 분산 처리로 확장성 확보
- 대규모 데이터 병렬 처리
- 수평적 확장을 통한 성능 향상

## 주요 사용 사례

### 1. 네트워크 장애 신속 대응

네트워크 장애 발생 시 실시간 모니터링 데이터와 AI 기반 RCA를 통해 신속하게 근본 원인을 파악하고, 자동 알람 연계를 통해 빠른 복구 조치를 수행합니다.

### 2. 트래픽 성능 최적화

sFlow 기반 트래픽 분석을 통해 네트워크상의 병목 지점을 식별하고, SDN 기반 트래픽 엔지니어링으로 최적의 경로를 설정하여 전체 네트워크 성능을 개선합니다.

### 3. 광학망 신뢰성 향상

광학 네트워크의 품질 파라미터를 지속적으로 모니터링하고, 예측 유지보수 기능으로 장비 장애를 사전에 예방하여 광학망의 가용성을 극대화합니다.

### 4. 통합 네트워크 관리

IP, 광학, 무선 등 이질적인 네트워크를 통합된 플랫폼에서 관리하여 운영 효율을 높이고, 복잡도를 낮추며, 일관된 모니터링과 제어를 가능하게 합니다.

### 5. 용량 계획 및 의사결정 지원

장기간의 성능 추이 분석과 AI 기반 예측 모델을 활용하여 미래 트래픽을 예측하고, 인프라 확장 시점과 규모를 과학적으로 결정할 수 있도록 지원합니다.

## 서브프로젝트 개요

### 1. nia-server-2021 (핵심 엔진 플랫폼)

**목적**: 네트워크 데이터 수집, 전처리, 분석, 알람 생성 등 핵심 처리를 담당하는 마이크로서비스 플랫폼

**주요 기능**:
- 21개의 독립적인 Spring Boot 마이크로서비스로 구성
- 데이터 정규화, 성능 분석, 알람 처리, 근본 원인 분석
- RabbitMQ 기반 이벤트 기반 통신 아키텍처
- SDN, EMS 등 외부 시스템 연동

**기술 스택**:
- Spring Boot 2.3.8~2.7.17
- Java 8-11
- Maven 빌드 도구
- RabbitMQ/AMQP 메시징
- PostgreSQL 데이터베이스

### 2. nia_optical_pm (광학 네트워크 예측 유지보수)

**목적**: 광학 네트워크 성능 모니터링 및 ML 기반 예측 유지보수

**주요 기능**:
- TL1/ROADM 기반 광학 장비 제어 및 관리
- 광학 성능 파라미터 분석
- 머신러닝 기반 장애 예측
- 포트 성능 예측 및 유지보수 계획

**기술 스택**:
- Spring Boot 3.5.8
- Java 17
- Maven 다중 모듈 구조
- RabbitMQ 서비스 간 통신

### 3. Engine_nia_ip_sdn_oproute (IP-SDN 라우팅 관리)

**목적**: SDN 기반 IP 네트워크 라우팅 최적화 및 제어

**주요 기능**:
- IP 네트워크 라우팅 정책 관리
- SDN 기반 트래픽 엔지니어링
- 네트워크 토폴로지 관리
- SSH/NETCONF 기반 장비 제어

**기술 스택**:
- 백엔드: Spring Boot 2.5.3, Java 11
- 프론트엔드: Vue 3.2, Bootstrap Vue 3
- 데이터베이스: PostgreSQL
- 빌드: Maven(백엔드), Webpack(프론트엔드)

### 4. web_2024 (2024 웹 대시보드 플랫폼)

**목적**: NIA 핵심 엔진의 데이터를 시각화하고 조작하는 통합 웹 인터페이스

**주요 기능**:
- 네트워크 성능 대시보드 및 모니터링
- 트래픽 분석 및 시각화
- 알람 관리 및 처리 UI
- 다양한 네트워크 도메인 통합 관리

**기술 스택**:
- 프론트엔드: Vue 2.6.11, Element UI, ag-Grid, ECharts, Cytoscape
- 백엔드: Spring Boot 2.7.17, Java 8, Gradle
- 빌드: Gradle 다중 모듈, FE dist를 BE static 리소스로 통합

## 기술적 특징

### 마이크로서비스 아키텍처

- 21개의 독립적인 서비스로 기능 분리
- RabbitMQ 기반 느슨한 결합(Loose Coupling)
- 수평적 확장 가능한 설계
- 각 서비스의 독립적인 배포 및 운영

### 이벤트 기반 통신

- RabbitMQ를 통한 비동기 메시징
- 서비스 간 느슨한 결합으로 복원력 향상
- 메시지 기반 워크플로우로 복잡한 비즈니스 로직 구현

### 데이터베이스 전략

- PostgreSQL을 중앙 데이터 저장소로 사용
- JPA/MyBatis ORM을 통한 데이터 접근 추상화
- 스케일링을 고려한 파티셔닝 및 인덱싱 전략

### 보안

- Spring Security 기반 인증/인가
- Jasypt를 통한 민감한 데이터 암호화
- BouncyCastle 라이브러리를 통한 SSL/TLS 지원
- 안전한 외부 연동 프로토콜 지원
