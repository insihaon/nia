# 순단장애/예지보전 네이밍 규칙 정리 — 1단계 (폴더/파일)

> 작업일: 2026-04-08
> 범위: Store 모듈 리네이밍 + 미사용 파일 삭제

---

## 1. Store 모듈 리네이밍

### 변경 사유

- `untact.js`는 다른 프로젝트 시스템의 명칭으로, 현재 NIA 프로젝트의 도메인과 무관
- 실제 역할: 순단장애(MBA)와 예지보전(PMM)의 RCA 티켓 상태 관리

### 변경 내용

| 구분 | 변경 전 | 변경 후 |
|------|---------|---------|
| Store 모듈 파일 | `store/modules/untact.js` | `store/modules/rcaTicket.js` |
| Vuex 네임스페이스 | `untact` | `rcaTicket` (require.context 자동 등록) |

### 참조 업데이트 파일 목록

| 파일 | 변경 내용 |
|------|----------|
| `TransientOutageList.vue` | `untact/insertTicketMbaList` → `rcaTicket/insertTicketMbaList`, `this.untact.xxx` → `this.rcaTicket.xxx` (3곳) |
| `PredictiveMaintenanceTicket.vue` | `untact/insertTicketPmmList` → `rcaTicket/insertTicketPmmList`, `this.untact.xxx` → `this.rcaTicket.xxx` (2곳) |
| `ModalPredictiveReviewOpinion.vue` | `this.untact.authority` → `this.rcaTicket.authority` (2곳) |
| `topasFilter.js` | `untact/changeSelectedSingleButton` → `rcaTicket/changeSelectedSingleButton` (1곳) |
| `Base.js` | `untact: (state) => state.untact` → `rcaTicket: (state) => state.rcaTicket` (1곳) |
| `webSocketManager.js` | `const { untact }` → `const { rcaTicket }`, getter 참조 변경 (2곳) |
| `variables.scss` | 주석 `//untact` → `// rcaTicket (순단장애/예지보전)` (2곳) |

### 변경 제외 대상

| 파일 | 사유 |
|------|------|
| `map2d_untact.min.js` | 외부 번들 라이브러리, 파일명 변경 불가 |
| `webSocketManager.min.js` / `global.min.js` / `Base.min.js` | 빌드된 min 파일 (원본 .js만 수정) |
| `api/nia.js` | `// untact code` 주석만 존재, 기능 영향 없음 |
| `webSocketManager.js` 176행 | `project === 'untact'` — 서버에서 내려오는 프로젝트 식별자이므로 변경 불가 |

## 4. 기존 네이밍 컨벤션 정리

| 레이어 | 규칙 | 예시 |
|--------|------|------|
| 페이지 폴더 | PascalCase | `TransientOutage`, `PredictiveMaintenance` |
| Vue 컴포넌트 | PascalCase | `TransientOutageList.vue`, `PredictiveMaintenanceTicket.vue` |
| 모달 컴포넌트 | `Modal` + PascalCase | `ModalRcaBatchFix.vue` |
| Store 모듈 | camelCase | `rcaTicket.js`, `systemMonitoring.js` |
| Store Actions | camelCase + 동사 접두사 | `insertTicketMbaList`, `deleteTicketPmmList` |
| API 함수 | camelCase + `api` 접두사 | `apiRcaRequest` |
| 라우트 경로 | camelCase | `/transientOutage/transientOutageList` |
| SCSS 변수 | $kebab-case | `$nia-primary`, `$miniFontSize` |
